Selected Publications

ICML Oral

SparseTSF

Introduced a cross-period sparse forecasting technique that dramatically reduces model parameters (down to ~1,000) while strengthening long-horizon dependency modeling. As an implicit regularization mechanism, it improves both robustness and generalization. The approach has been applied as a core model in Didi ride-hailing demand-supply forecasting.

NeurIPS Spotlight

CycleNet

Proposed explicit periodic-pattern modeling for time-series forecasting. Through cycle-residual decomposition, the model first captures periodic components and then models residual components more effectively, improving predictive accuracy and interpretability.

ICML

TQNet

Proposed temporal query learning for multivariate forecasting. By modeling global periodic patterns, TQNet offers more robust dependency learning across variables and better overall multivariate forecasting performance.
