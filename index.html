Selected Publications

ICML Oral

SparseTSF

Proposed a cross-period sparse forecasting framework that dramatically reduces model size (as low as 1,000 parameters) while improving long-range dependency modeling. The sparse design also acts as implicit regularization, enhancing robustness and generalization. The method has been deployed as a core model in Didi’s ride-hailing supply-demand forecasting workflow.

NeurIPS Spotlight

CycleNet

Introduced explicit periodic-pattern modeling for time-series data to better utilize cyclical structure. Through a cycle-residual decomposition strategy, the model captures periodic components first and then learns residuals more effectively, improving both forecasting performance and interpretability.

ICML

TQNet

Proposed a temporal query mechanism that models global periodic patterns to support more robust multivariate dependency learning, leading to improved multivariate forecasting ability.
